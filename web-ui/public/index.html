ℹ️ Quick Reference

Command Descriptions
  • Check Status: List all launchers and install status
  • Show Installers: Find installers in ./installers/ directory
  • Quick Setup: Reinitialize dependencies + install missing apps
  • Full Setup: Purge + init + install all (fresh start)
  • Launch [App]: Start a launcher with Proton
  • Unmount Z: Hide /mnt from Wine (fixes disk space warnings)
  • Mount Z: Restore /mnt access to Wine
  • Suppress Warnings: Mute Z: drive error logs
Troubleshooting
  • Installer won't download? Check INSTALLER_LINKS.md for manual download
  • Launcher won't launch? Click "Check Status" to verify installation
  • Disk space warnings in EA Desktop? Try "Unmount Z:" first
  • GOG Galaxy update fails? Use "Suppress Warnings" then "Unmount Z:"
  • Server not responding? Check if Node.js is running: npm start
